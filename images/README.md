## Instructions

1. **Add your photos** to this folder with these names:
   - `photo1.jpg`
   - `photo2.jpg`
   - `photo3.jpg`
   - `photo4.jpg`
   - `photo5.jpg`
   - `photo6.jpg`

2. **Recommended Image Specs**:
   - Format: JPG, PNG, or WebP
   - Size: At least 800x800 pixels (square format works best)
   - File size: Keep under 500KB for best performance
   - Quality: High quality but compressed (use TinyPNG.com)

3. **Naming Convention**:
   - Use lowercase names
   - No spaces in filenames
   - Use underscores or dashes: `photo_1.jpg` or `photo-1.jpg`

## Converting to Base64 (Optional)

If you want to embed images directly in HTML:

1. Make sure all your photos are in this folder
2. Open terminal in the project root
3. Run:
   ```bash
   cd js
   node convertImagesToBase64.js
   ```
4. Follow the instructions in the output
